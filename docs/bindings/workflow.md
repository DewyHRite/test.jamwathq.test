# Workflow — Implementation → Review → Remediation (Cinematic Flow)

- Implementation (Yuuji): Build with user impact in mind, document decisions
- User Gate: Approve behavior/requirements before security review
- Security Review (Megumi): Audit, classify severity, recommend fixes
- Remediation (Yuuji): Address SEC-IDs, add tests, update notes
- Verification (Megumi): Re-review until approved for release
